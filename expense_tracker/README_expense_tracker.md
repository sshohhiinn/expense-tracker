## 📋 Project Description

A full-featured **Expense Tracker** application built with Python that allows users to track their daily expenses. Data is stored in a **CSV file** for easy access and export. Available in both **Desktop GUI (Tkinter)** and **Web App (Flask)** versions.



## ✨ Features

- ➕ Add expenses with **Date, Category, Description & Amount**
- 📋 View all expenses in a **clean table**
- ✏️ Edit any existing expense
- 🗑️ Delete expenses by ID
- 🔍 Search & filter by **Category, Month, Date or Keyword**
- 📊 Summary with **statistics & category breakdown**
- 📤 Export monthly report as **CSV file**
- 💾 Auto-saves all data to `expenses.csv`
